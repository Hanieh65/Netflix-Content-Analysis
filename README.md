# Netflix Content Analysis (SQL + Power BI)

## Project Overview

This project explores and analyses a Netflix dataset using SQL for data cleaning and transformation, and Power BI for dashboard creation and visualization.

The dataset includes information about Movies and TV Shows available on Netflix between 1975 and 2021.

The goal of this project is to demonstrate an end-to-end data analysis workflow:

- Data cleaning using SQL
- Data preparation for analytics
- Insight generation
- Dashboard creation using Power BI

---

## Dataset

Dataset: Netflix Titles Dataset (CSV)

The dataset contains:

- Title
- Type (Movie / TV Show)
- Release year
- Genre
- Duration
- Country
- Date added to Netflix

Cleaning steps:

- checked for duplicate records
- Standardized duration format
- Converted date columns
- Structured genre categories
- Prepared dataset for visualization

---

## Tools Used

- SQL
- Power BI
- CSV dataset

---

## Key Insights

Main insights from the dashboard:

- Movies represent 55.11% of content
- TV Shows represent 44.89%
- Average Movie Duration: 104.37 minutes
- Average TV Show Length: 2.23 seasons
- Content production increased significantly after 2015
